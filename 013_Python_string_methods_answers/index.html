    and split it into a list of sub-strings. * Then re-construct the string, separating each word by a colon character ':' * Print out the list of sub-strings and the re-constructed string

    # Answer

# Take the string
string = '''
The Owl and the Pussy-cat went to sea 
In a beautiful pea-green boat, 
They took some honey, and plenty of money, 
Wrapped up in a five-pound note.
'''

# and split it into a list of sub-strings.
list_string = string.split()
# print this out
print(list_string)

# Then re-construct the string, separating each word by a colon character ':'
recon_string = ':'.join(list_string)
# print this out
print(recon_string)
